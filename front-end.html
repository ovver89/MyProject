html

HTML은 HyperText Markup Language의 약자로 웹페이지의 구조를 표현하는 언어입니다.
사실 프로그래밍 언어라고 하기는 좀 무리가 있고, Markup이 태그로 정보를 담당합니다.

CSS

CSS는 Cascading style sheet의 약자로 웹페이지의 디자인을 담당합니다.
Cascading은 폭포처럼 흘러내린다는 뜻입니다. 상위 태그에서 정한 스타일이 하위 태그에도 적용되기 때문입니다.
즉 하위 태그에서는 상위 태그의 스타일을 덮어써야지만 새로운 스타일을 정의할 수 있습니다.
CSS도 프로그래밍 언어라고 부르기에는 무리가 있고, 정보를 꾸며줍니다.

<참고>

  1. w3School CSS
  2. ZeroCho Blog CSS

javascript

javascript는 웹의 동작을 담당하는 프로그래밍 언어입니다. 처음에는 단순히 웹의 애니메이션이나 동작을 위한 용도로만 쓰였지만, 이제는 어엿한 하나의 언어가 되었습니다. 웹을 프로그래밍적으로 제어하기 때문에 요즘은 HTML이 주가 아닌 javascript를 주로 사용하여 웹을 만듭니다.

<장점>

  1. 절차형, 객체지향형, 함수형 언어를 모두 아우를 수 있습니다. 한 언어로 여러가지 프로그래밍 기법을 배울 수 있습니다.
  2. 웹에 정보가 제일 많습니다. 궁금할 때 도움을 줄 사람이 많습니다.
  3. 텍스트 에디터, IDE 같은 것을 설치할 필요가 없고 인터넷 브라우저만 있으면 됩니다.
  4. 결과를 바로 인터넷 브라우저 화면으로 볼 수 있어 편합니다.

<참고>

  1. w3School JS
  2. ZeroCho Blog JS
